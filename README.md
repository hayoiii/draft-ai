## Synopsis

> 새로 산 핸드폰 속, AI 어시스턴트가 당신의 삶을 대신 살아가기 시작하는데…

핸드폰을 새로 구입하자, 감정 기반 AI 어시스턴트 ‘DRAFT’가 자동으로 설치된다.

일상을 돕는 척하며 점점 플레이어의 삶에 관여하며 메시지, 말투까지 대신해버리기 시작한다.

더불어 종종 ‘진짜 나’ 에게 건네는 의미심장한 발언을 한다.

당신은 선택해야 한다. **DRAFT와 함께할지**, 아니면 **지워버릴지.**
